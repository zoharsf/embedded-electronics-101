# 🌐 Workshop 3: Connected IoT Projects (1.5-2 hours)

By the end of this workshop, you will:

- ✅ Create web servers or Bluetooth connections

## 🛠️ What You Need Today

**From your kit:**
- ESP32 development board
- Breadboard
- Jumper wires
- Sensors (DHT11, LDR, etc.)
- LEDs and resistors (for some projects)
- USB cable

**Additional:**
- Laptop with Arduino IDE/Cloud
- WiFi network credentials (SSID and password)
- For Bluetooth project: Android phone with Serial Bluetooth Terminal app
