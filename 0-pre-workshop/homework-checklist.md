## Quick Knowledge Check (Optional)

Answer these in your head (no need to submit):

1. **What are the 3 core concepts?**
   - Answer: Voltage (push), Current (flow), Resistance (brake)

2. **LED polarity: which leg is positive?**
   - Answer: Long leg = positive

3. **Why do we use resistors with LEDs?**
   - Answer: Limit current so LED doesn't burn out

4. **What does `pinMode(2, OUTPUT)` do?**
   - Answer: Sets pin 2 to output mode (we'll control it)

5. **What's the difference between setup() and loop()?**
   - Answer: setup() runs once, loop() runs forever
