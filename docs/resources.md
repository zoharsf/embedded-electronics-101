## üìñ Glossary of Terms {#glossary}

Common electronics and programming terms explained simply.

**Key Concepts:**
- **Voltage (V)** - Electrical pressure, measured in volts
- **Current (A)** - Flow of electricity, measured in amps
- **Resistance (Œ©)** - Opposition to current, measured in ohms
- **GPIO** - General Purpose Input/Output pins
- **PWM** - Pulse Width Modulation for controlling brightness/speed
- **I2C** - Communication protocol for sensors
- **Sketch** - Arduino program

[View complete glossary](https://github.com/zoharsf/embedded-electronics-101/blob/main/resources/glossary.md)
