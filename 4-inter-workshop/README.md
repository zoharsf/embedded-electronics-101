# 📡 Inter-Workshop 2 Homework (45-60 min)

Great job at Workshop 2! Ready to go wireless?

Workshop 3 introduces **connectivity** - WiFi, Bluetooth, and web servers. This homework prepares you for building connected IoT projects.

## What You'll Learn

1. **WiFi Basics** (15 min read)
   - How ESP32 connects to WiFi
   - Network scanning and connection
   - IP addresses and clients

2. **Bluetooth Basics** (15 min read)
   - Bluetooth Classic vs BLE
   - Serial communication over Bluetooth
   - Mobile app control

3. **Web Server Fundamentals** (15 min read)
   - HTTP requests and responses
   - HTML basics for dashboards
   - REST APIs

4. **Choose Your Workshop 3 Project** (5 min)
   - Pick ONE of 3 advanced projects

---

## The 3 Project Options

### 📡 WiFi Weather Station
Host a web page showing real-time sensor data. Access from any device on your network!
**Difficulty:** ⭐⭐

### 🎮 Bluetooth Controller
Control LEDs and motors from your phone via Bluetooth.
**Difficulty:** ⭐⭐

### 🌐 Web Dashboard
Full IoT dashboard with data logging, charts, and remote control.
**Difficulty:** ⭐⭐⭐
