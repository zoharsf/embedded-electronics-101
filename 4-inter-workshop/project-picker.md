# 🎯 Pick Your Workshop 3 Project

Choose **ONE** project. You'll work on it during Workshop 3 to build a connected IoT device!

---

## 📡 Project 1: WiFi Weather Station

**What you build:** A web-based weather station that displays temperature, humidity, and light levels on a webpage accessible from any device on your network.

**What you learn:** 
- WiFi connectivity
- Web server basics
- HTML dashboards
- Real-time data display

**Cool factor:** Access your sensor data from your phone, tablet, or computer - no app needed!

**Difficulty:** ⭐⭐ Medium

**Take-home:** A working IoT weather station

**Perfect if you:**
- Want to learn web development basics
- Like data visualization
- Want something practical for home use

---

## 🎮 Project 2: Bluetooth Controller

**What you build:** Control LEDs, motors, or servos from your phone via Bluetooth. Create a custom control interface!

**What you learn:**
- Bluetooth Serial communication
- Mobile app integration
- Command protocols
- Wireless hardware control

**Cool factor:** Control your projects wirelessly from your pocket!

**Difficulty:** ⭐⭐ Medium

**Take-home:** A Bluetooth-controlled device

**Perfect if you:**
- Have an Android phone (iOS doesn't support Bluetooth Classic well)
- Want direct control without WiFi
- Like mobile app development
- Want to build robot controllers

---

## 🌐 Project 3: Full IoT Dashboard

**What you build:** Complete IoT system with data logging, real-time charts, and remote control via web interface. Most advanced project!

**What you learn:**
- Advanced web server techniques
- JavaScript for interactivity
- JSON APIs
- Real-time data streaming
- CSS styling

**Cool factor:** Professional-grade IoT dashboard running on a $10 chip!

**Difficulty:** ⭐⭐⭐ Harder

**Take-home:** Enterprise-level IoT system

**Perfect if you:**
- Want a challenge
- Have web development experience
- Want to build a portfolio project
- Like full-stack development
