    🎯 Pick Your Workshop 3 Project¶

    Choose ONE project. You'll work on it during Workshop 3 to build a connected IoT device!

    📡 Project 1: WiFi Weather Station¶

    What you build: A web-based weather station that displays temperature, humidity, and light levels on a webpage accessible from any device on your network.

    What you learn: - WiFi connectivity - Web server basics - HTML dashboards - Real-time data display

    Cool factor: Access your sensor data from your phone, tablet, or computer - no app needed!

    Difficulty: ⭐⭐ Medium

    Take-home: A working IoT weather station

    Perfect if you: - Want to learn web development basics - Like data visualization - Want something practical for home use

    🎮 Project 2: Bluetooth Controller¶

    What you build: Control LEDs, motors, or servos from your phone via Bluetooth. Create a custom control interface!

    What you learn: - Bluetooth Serial communication - Mobile app integration - Command protocols - Wireless hardware control

    Cool factor: Control your projects wirelessly from your pocket!

    Difficulty: ⭐⭐ Medium

    Take-home: A Bluetooth-controlled device

    Perfect if you: - Have an Android phone (iOS doesn't support Bluetooth Classic well) - Want direct control without WiFi - Like mobile app development - Want to build robot controllers

    🌐 Project 3: Full IoT Dashboard¶

    What you build: Complete IoT system with data logging, real-time charts, and remote control via web interface. Most advanced project!

    What you learn: - Advanced web server techniques - JavaScript for interactivity - JSON APIs - Real-time data streaming - CSS styling

    Cool factor: Professional-grade IoT dashboard running on a $10 chip!

    Difficulty: ⭐⭐⭐ Harder

    Take-home: Enterprise-level IoT system

    Perfect if you: - Want a challenge - Have web development experience - Want to build a portfolio project - Like full-stack development

    What Each Project Includes¶

    All Projects Come With:¶

    • ✅ Complete circuit diagrams
    • ✅ Fully commented code
    • ✅ Step-by-step build instructions
    • ✅ Troubleshooting guide
    • ✅ Extension ideas

    WiFi Weather Station:¶

    • Temperature/humidity sensor code
    • HTML dashboard
    • Auto-refresh display
    • Responsive design (works on phone/tablet)

    Bluetooth Controller:¶

    • Bluetooth Serial setup
    • Command protocol
    • LED/motor control examples
    • Mobile app recommendations

    IoT Dashboard:¶

    • Multi-sensor integration
    • JavaScript real-time updates
    • JSON API
    • Modern CSS design
    • Chart visualization

    How to Choose¶

    Choose WiFi Weather Station if:¶

    • ✅ You want to learn web basics
    • ✅ You want something practical for home
    • ✅ You like seeing data
    • ✅ You have WiFi available

    Choose Bluetooth Controller if:¶

    • ✅ You have an Android phone
    • ✅ You want direct control
    • ✅ You don't have WiFi available
    • ✅ You want to build robots/RC projects

    Choose IoT Dashboard if:¶

    • ✅ You want maximum challenge
    • ✅ You know some HTML/CSS already
    • ✅ You want a portfolio project
    • ✅ You want the "wow" factor

    Still Unsure?¶

    Based on your Workshop 2 project:

    • Built Sensor Logger? → WiFi Weather Station (natural progression)
    • Built LED Light Show? → Bluetooth Controller (similar control concepts)
    • Built Button Game? → Bluetooth Controller (interactive control)
    • Built Data Dashboard? → IoT Dashboard (level up your dashboard!)

    After Workshop 3¶

    All three projects can be extended:

    WiFi Weather Station → - Add more sensors - Log data to SD card - Cloud integration (ThingSpeak, Blynk) - Weather forecasting

    Bluetooth Controller → - Create custom mobile app - Add voice control - Multi-device control - Game controller

    IoT Dashboard → - User authentication - Database storage - Email/SMS alerts - Remote internet access

    Prerequisites Check¶

    Before Workshop 3, make sure you have: - ✅ Completed Workshops 1 & 2 - ✅ Read WiFi/Bluetooth/Web Server basics - ✅ Tested WiFi connection on your ESP32 - ✅ (For Bluetooth project) Android phone with Serial Bluetooth Terminal app - ✅ (For WiFi projects) Your network SSID and password ready

    Workshop 3 Groups¶

    We'll organize into 3 groups based on project choice: - Group A: WiFi Weather Station - Group B: Bluetooth Controller - Group C: IoT Dashboard

    Don't worry! You can always build the other projects at home later. All materials stay available!

    Ready? Remember your choice and see you at Workshop 3! 🚀
